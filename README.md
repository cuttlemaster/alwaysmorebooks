# Always More Books
Using basic HTML for structure and CSS for styles I wanted to mock up a proof of concept and/or wireframe for a blog I want to eventually build with React. Included with all of that I wanted to figure out my mobile and tablet experiences along with the usual desktop one so I wouldn't have to worry about making those decisions while learning React at the same time.

### Development Instructions
* When developing locally, run `npm run scss` to ensure styles are compiled as changes are made.
